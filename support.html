Navigating in Figma is simple once you know the controls:

  • Zoom in/out: Hold Ctrl (or ⌘ on Mac) and scroll your mouse wheel

Duplicating Files

Duplicating creates your own personal copy that you can fully edit.

  1. Open the file link we sent you
  2. Make sure you are signed into Figma
  3. Click the Figma logo (top-left corner) or go to File in the menu
  4. Click "Duplicate to your drafts"
  5. A new tab will open with your own copy — this is yours to edit!
Tip: Your duplicated file lives in your "Drafts" folder. You can always find it by going to figma.com and clicking "Drafts" in the left sidebar.

If you don't see the duplicate option, try these fixes:

  • Sign in first — You must be logged into a Figma account to duplicate
  • Look in the right place — Click the Figma logo (☰) in the top-left corner, then look for "Duplicate to your drafts"
  • Try right-clicking the file name in the toolbar at the top — a menu with duplicate should appear
  • Check if you already duplicated it — Look in your Drafts at figma.com
Note: If the file owner has restricted permissions, duplicating may be disabled. Reach out to us if this is the case.

Your duplicated file is in your Drafts folder.

  1. Go to figma.com
  2. In the left sidebar, click "Drafts"
  3. Your duplicated file will be at the top — it'll have the same name with "(Copy)" added
Tip: You can rename the file by double-clicking its name at the top of the editor, or right-clicking it in the Drafts list.

Editing Your File

Editing text is straightforward:

  1. Double-click on any text in the design
  2. The text will become editable — type your new content
  3. Click anywhere outside the text when you're done

If double-clicking doesn't work, you may need to double-click multiple times to "drill into" grouped layers. Keep double-clicking until the text highlights.

Tip: You can change the font, size, and color in the right sidebar under "Text" once the text is selected.

To swap out an image:

  1. Click on the image you want to replace
  2. In the right sidebar, look for the "Fill" section
  3. Click the image thumbnail in the fill section
  4. Click "Choose image" and select your new photo from your computer

Alternative method: Copy an image to your clipboard, select the image frame in Figma, and press Ctrl+Shift+V (or ⌘+Shift+V on Mac) to paste it as a fill.

  1. Click on the element you want to recolor
  2. In the right sidebar, find the "Fill" section
  3. Click the colored square to open the color picker
  4. Pick a new color or type in a hex code (e.g., #FF5733)
Tip: If you have a brand color, paste the hex code directly into the color field at the bottom of the picker — it's the fastest way.

If you see a ⚠️ warning icon next to text or fonts look different than expected, the file uses fonts you don't have.

Most fonts will load automatically in Figma since they include Google Fonts. But if a custom font is used:

  1. Look for a yellow warning banner at the top — click it to see which fonts are missing
  2. If we provided the font files, install them on your computer then refresh Figma
  3. Or download the Figma Font Helper (a small app that lets Figma access your installed fonts in the browser)
  4. After installing, refresh the page and fonts should appear correctly
Don't have the fonts? Message us and we'll send you the font files or suggest a free alternative.

Don't panic! Figma has unlimited undo:

  • Press Ctrl+Z (or ⌘+Z on Mac) to undo
  • Press Ctrl+Shift+Z to redo

You can also access Version History to go back to any previous state:

  1. Click the Figma logo (top-left)
  2. Go to File → Show Version History
  3. Browse through past versions on the right sidebar
  4. Click any version to preview it, then click "Restore this version"
Tip: Figma auto-saves every change, so your work is never lost. Version history lets you travel back in time to any point.

Elements in Figma are often grouped or layered. Here's how to get to the one you need:

  • Double-click repeatedly to drill into groups until you reach the specific element
  • Hold Ctrl (or ⌘) and click to select through groups directly
  • Use the Layers panel (left sidebar) to find and click on the exact layer
Tip: If something is behind another element, right-click and choose "Select layer" to pick from overlapping items.

Exporting & Downloading

  1. Select the frame or element you want to export (click on it)
  2. In the right sidebar, scroll down to the "Export" section
  3. Click the "+" button to add an export setting
  4. Choose your format: PNG (images), SVG (vector), PDF (documents)
  5. Click "Export [name]" to download
Tip: For high-quality images, change the export size to 2x or 3x before exporting. This gives you a higher resolution image.

To export multiple frames or the entire page:

  1. Select all the frames you want — use Ctrl+A (or ⌘+A) to select all
  2. Or hold Shift and click each frame you want
  3. Go to File → Export… (or press Ctrl+Shift+E)
  4. Review the export list, pick your format, and click "Export"
Tip: Export as PDF if you need the whole design in a single document (great for presentations or printing).
  1. Click on any element in the design
  2. In the right sidebar, look for the "Inspect" tab (or click the code icon </>)
  3. You'll see CSS properties, colors, fonts, and spacing values
  4. Click to copy any value

This is great for developers who need exact colors, font sizes, or spacing from the design.

Troubleshooting

Large design files can sometimes be demanding. Try these fixes:

  • Close other browser tabs — Figma uses a lot of memory
  • Use Chrome — It has the best performance with Figma
  • Zoom out — Rendering fewer details speeds things up
  • Avoid browser extensions — Ad blockers and other extensions can interfere
  • Restart your browser — A fresh start often fixes lag issues
Tip: If the file has a lot of pages, only work on the page you need. Switching between pages helps reduce memory usage.

Figma files can have multiple pages. To switch between them:

  1. Look at the top of the left sidebar (Layers panel)
  2. You'll see page names listed there (e.g., "Page 1", "Desktop", "Mobile")
  3. Click on a page name to switch to it

If you don't see the sidebar, click the Layers icon (stacked squares) on the left toolbar to open it.

If the file includes an interactive prototype (clickable demo), here's how to view it:

  1. Click the ▶ Play button in the top-right corner of the editor
  2. A new tab will open with the prototype view
  3. Click around to interact with the design — buttons, links, and navigation should work
  4. Press R to restart the prototype from the beginning
Tip: Blue highlighted areas show you what's clickable. If nothing seems to happen when you click, the prototype might not have interactions set up for that element.

Figma saves automatically! There's no save button because every change is saved in real time to the cloud.

You'll see a small cloud icon with a checkmark in the toolbar — that means everything is saved.

If you see a cloud with an X, it means Figma lost connection. Don't worry — your changes will sync once you're back online.

Tip: If you want to save a specific "checkpoint" of your design, go to File → Save to Version History and add a description. This is like creating a named bookmark you can return to later.

If it's locked:
